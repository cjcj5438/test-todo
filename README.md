# Todo App

A simple and beautiful todo list web application built with React + Vite.

## Features

- Add new todos
- Mark todos as completed/incomplete
- Delete todos
- View statistics (total, completed, remaining)
- Responsive design
- Beautiful UI with gradient background

## Tech Stack

- React 18
- Vite
- CSS3

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

### Build

```bash
npm run build
```

Builds the app for production to the `dist` folder.

### Preview Production Build

```bash
npm run preview
```
